# Google shopping E2E test
This project contains automated tests for five different test cases of Google shopping in Canada.

### Tech/framework used  
* Javascript
* Node.js
* Selenium webdriver 
* Cucumber
* Chai

### Prerequisites
To run the tests localy in you machine, you should install: 
 
`Node.js >= v7.6` 

### Running the tests  
```
npm install

npm run test  

```
